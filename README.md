## Getting Started  

Activate the virtual environment  
```bash    
source env/bin/activate
```  
  
Run the FastAPI application with Uvicorn.  
```bash
uvicorn app.main:app --reload 
```  
  
Stop the application
```bash
CTRL + C  
```
